# summer-migration-project
### Does site selection bias have a lesser impact on migratory species?
###### Since migratory species travel to different sites for reproduction and different climates (dependent on the type of migratory species), I believe that site selection bias will have a lesser impact on studies done on migratory species since they are traveling more frequently (thus spending less time in a place) which means they have less of an impact on the non-migratory species and the environment in the site selected for sampling.  ​
Goal: to use data analysis in python to investigate a data set - see bottom fish trawl data on NOAA
